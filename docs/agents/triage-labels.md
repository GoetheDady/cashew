# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the actual label strings used in this repo's issue tracker.

| Label in mattpocock/skills | Label in our tracker | 中文名称 | Meaning |
| -------------------------- | -------------------- | -------- | ------- |
| `needs-triage` | `needs-triage` | 待分类 | Maintainer needs to evaluate this issue |
| `needs-info` | `needs-info` | 待补充信息 | Waiting on reporter for more information |
| `ready-for-agent` | `ready-for-agent` | Agent 可处理 | Fully specified, ready for an AFK agent |
| `ready-for-human` | `ready-for-human` | 需人工处理 | Requires human implementation |
| `wontfix` | `wontfix` | 不处理 | Will not be actioned |

When a skill mentions a role (e.g. "apply the AFK-ready triage label"), use the corresponding label string from this table.
